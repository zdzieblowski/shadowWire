# shadowWire
SECURE COMMUNICATION NETWORK

### CLIENT DEPENDENCIES:
```
pip install cryptography requests
```

### CLIENT USAGE:
```bash
# INITIALIZE
python initialize.py [-a ALIAS]

# RECEIVE
python decode.py [-a ALIAS]

#TRANSMIT
python encode.py -r RECIPIENTS_PUBLIC_KEY [-a ALIAS]
```

### SERVER DEPENDENCIES:
```
npm ci
```
